Szenarioauswahl
Auswahl eines Szenarios, um das Lernmodul mit einem vordefinierten Beispiel zu initialisieren.
Schrittauswahl
Auswahl des nächsten Schritts, den das Lernmodul machen soll:
  • Mit '|<' kann ein Schritt bis zum Beginn ausgeführt werden.
  • Mit '<<' (wenn verfügbar) kann ein großer Schritt rückwärts ausgeführt werden.
  • Mit '<' kann ein kleiner Schritt rückwärts ausgeführt werden.
  • Mit 'O' kann das Lernmodul zurückgesetzt werden.
  • Mit '>' kann ein kleiner Schritt vorwärts ausgeführt werden.
  • Mit '>>' (wenn verfügbar) kann ein großer vorwärts rückwärts ausgeführt werden.
  • Mit '>|' kann ein Schritt bis zum Ende ausgeführt werden.
Erläuterung zur Unterscheidung der Schritte:
  • Mit kleinen Schritten können Details des Lerninhalts betrachtet werden, um das konkrete Vorgehen des Lerninhalts nachzuvollziehen.
  • Mit großen Schritten können Details des Lerninhalts übersprungen werden, um die Zusammenhänge des Lerninhalts nachzuvollziehen.
